# Examples

Illustrative, compile-checked snippets organized to mirror `lib/`. The Flutter
and widget examples are meant to be read and adapted into an app rather than run
standalone; the `bloc` example is pure Dart and runs with `dart run`.

| Area | Example |
| --- | --- |
| bloc change predicates | [`bloc/bloc_utils_example.dart`](bloc/bloc_utils_example.dart) |
| Date/time extensions | [`time_utils/date_time_extensions_example.dart`](time_utils/date_time_extensions_example.dart) |
| Widget & `State` extensions, snackbars | [`flutter/flutter_extensions_example.dart`](flutter/flutter_extensions_example.dart) |
| Currency/number input formatting | [`flutter/number_format_input_example.dart`](flutter/number_format_input_example.dart) |
| form_builder helpers & stepper form | [`form_builder/form_builder_example.dart`](form_builder/form_builder_example.dart) |

`sfere` is covered in the API docs; a fuller example is coming in a later
release.
